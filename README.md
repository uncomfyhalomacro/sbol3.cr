# sbol3

An attempt to implement SBOL3 specification in Crystal.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     sbol3:
       github: uncomfyhalomacro/sbol3.cr
   ```

2. Run `shards install`
